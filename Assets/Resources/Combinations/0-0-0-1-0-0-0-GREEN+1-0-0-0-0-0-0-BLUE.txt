Rozdział III. Uderza w Was wizja. Widzicie cały Las z lotu sokoła, otoczony zewsząd mgłą. Knieja widzi, słyszy, czuje. Kiedyś nawet staczała bitwy... Na jedną krótką chwile łączycie się duszą z Lasem... te wrażenie niemal ścina Was z nóg. Las od teraz Was zna i nie będzie Wam wrogiem. Macie wrażenie, że jeden tutejszy Umysł wyjątkowo pilnie się Wam przygląda...

NIE CZYTAJ NA GŁOS. Odetchnij głęboko i zapewnij swoich pobratymców, że Las nie będzie chciał Was zabić, o ile okażecie mu szacunek.
