Rozdział V. Niemożliwe staje się możliwe! Wspaniała to bitka, odpieracie Drzewa i ich stęchłą magię, choć zostaje Was tylko trójka. Musicie zniszczyć tylko ostatnie, największe Drzewo... Wódz rusza pierwszy... i nagle pada. Potęzny młot rzucony z koron drzew miażdzy mu twarz, a mózg ścieka po zbroi. Lewitując, na ramionach ostatniego z Drzew ląduje mag w czerwonym stroju. Potężnymi czarami zamienia Was w brzozy, Wy jednak oponujecie. Nagle świat staje się czarny. Znacie już  Waszego oprawcę. Nie będziecie nigdy częścią Jego włości, prędzej wybierzecie nicość. Zostają z Was tylko połamane, ścięte brzozy.
