Rozdział III. Odkrywacie powoli, że Drzewa same ze sobą są bardzo skonfliktowane. Mają zaszłości sięgające setek lat, a niektóre kłócą się o wspólną przyszłość Lasu. Niektóre wspominają też dawną bitwę i ostrzegają... CZEKAJ!
