Rozdział IV. Udało się Wam w końcu znaleźć słabe punkty w systemie obronnym Lasu i umiecie wykorzystać go do własnych celów. Lepiej też rozumiecie istotę sporów wśród drzew – niektóre chcą otwarcia Lasu, niektóre szczelnego Jego zamknięcia. Jedne chcą obcych przyjmować, inne zabijać... Ale po co Drzewom plany na przyszłość? I skąd tak rozwinięte myśli w jaźniach... roślin?
